# Melius Homebrew tap

Homebrew formulae for [Melius](https://melius.com) tools.

## `mel` — the Melius CLI

```bash
brew install melius-ai/tap/mel
```

or

```bash
brew tap melius-ai/tap
brew install mel
```

`mel` is also on npm (`npm install -g @melius-ai/cli`). Docs: https://docs.melius.com/cli/overview
